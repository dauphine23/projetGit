Commandes GIT

  • GIT STATUS:
    voir les fichiers en attente d'ajout 
  • GIT ADD FICHIER:
    ajouter un fichier à l'index 
  • GIT ADD . :
    ajouter tous les fichiers modifiés à l'index
  • GIT COMMIT -m :
    "commentaire": fair un commit 
  • GIT DIFF:
    voir les différences entre l'index et le fichier
  • GIT COMMIT-a-m:
    Fait un git add et un git commit en même temps 
  • GIT rm -f fichier.txt:
    supprimer un fichier indexé de GIT
  • GIT RESET HEAD :
    suppression du dernier ajout avant commit
  • GIT RESET HEAD^ fichier.txt:
    Suppression du dernier commit pour le fichier
  • GIT checkout HEAD^fichier.txt
    Suppression de dernier commit du fichier
  • GIT LOG
    Afficher tous les logs
  • GIT LOG-3
    Afficher les messages des 3 deriniers commit
  • GIT LOG oneline:
    Afficher les logs sur une seule ligne
  • GIT CHECKOUT 2c3f4567 index.html
    Revient à une version antérieure du fichier
  • GIT CHECKOUT master index.html
    Annule le dernier checkout
  • GIT ADD REMOTE origin https://github.com/depot.git
    Synchronisation avec un dépot distant
  • GIT PULL
    Récupérer le dépot distant A faire avant tout push!!!
  • GIT PUSH origin master:
    Envoie des derniers commits sur le dépot distant 
  • GIT REVERT:
    Annulation du dernier push
  • GIT BRANCH
    voir les branches
  • GIT BRANCH ma-branche
    création d'une branche
  • GIT CHECKOUT ma-branche
    connexion à une branche
  • GIT CHECKOUT -b ma-branche
    Création et connexion à une branche
